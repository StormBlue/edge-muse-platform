# Internal Demo Script

1. Login as `sysadmin@example.com`.
2. Open 系统看板 and review KPI, provider share, trend, and Top users.
3. Create or edit a provider key, then open 管理员 and assign the key/quota to an admin.
4. Login as admin, create a user with an initial password, grant quota, disable/enable the user, and reset the password manually.
5. Login as user, create a text-to-image task, open the image viewer, copy prompt, and download.
6. Upload a reference image and run 图生图.
7. Switch to 对话 and verify recent context continuity.
8. Open 历史 and sort by 最近, 最早, and 任务最多.
9. Login as sysadmin and open 会话审计 for the user ID.
10. Review R2 private image access through `/api/i/:imageId`.
