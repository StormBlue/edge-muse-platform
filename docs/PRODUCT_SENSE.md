# 产品与角色观感

短文描述「谁为了什么而来」；字段级权限与技术细节见 [`USER_GUIDE.md`](./USER_GUIDE.md)、[`SECURITY.md`](./SECURITY.md)。

## 角色

| 角色                  | 目标                              | 主要产品面                                                                                                               |
| --------------------- | --------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| 普通用户 `user`       | 在配额内生图、看历史、复用 prompt | 「AI 图像生成」`/ai-image`、工作台 `/workspace`、历史、大图与资料（入口是否展示由 [`EXPERIMENTS.md`](./EXPERIMENTS.md)） |
| 管理员 `admin`        | 维系名下终端用户                  | 开户、配额、启停用户、重置密码、用量视图                                                                                 |
| 系统管理员 `sysadmin` | 平台配置与风控                    | Provider 密钥、管理员、生成入口、`/sysadmin/generation-entry`、案例库、公告、会话审计、系统偏好等                        |

无公开注册、无自助找回密码链路；账号与密钥由人工开通。

## 核心旅程（用户）

1. 登录 → 按 `generationEntry` 进入默认创作页或可切换入口。
2. 撰文生图或图生图 → 异步任务 → WebSocket/Web 合并状态查看结果。
3. 从历史会话复打开、重试或软删会话（规则见 [`API.md`](./API.md)）。

## 指标与实验

- 生成漏斗与页面用量：**结构化事件**，非多臂随机实验；管理与窗口见 `/sysadmin/generation-entry`。
